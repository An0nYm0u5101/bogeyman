
-- Introduction --


-- Header --
     8bits
    +-----+------------+
    | CMD | PARAMETERS |
    +-----+------------+

-- Commands --

    00 (Create Stream) - params: <stream_id><hostname><port>
        Creates a new stream.

    01 (Data Sync) - params: <stream_id><data>
        Starts data synchronization between the stream sides.

    03 (Close Stream) - params: <stream_id>
        Closes stream.

    04 (Set Stream Status) - params: <stream_id><status>
        Set the stream status in the other side of the connection.

    # This commands reserved for tunnel uses.
    05 (Ping) - No parameters
        Send a ping command. This command expect a Pong command, else It assume the connection is down.

    06 (Pong) - No parameters
        Ping response command.
